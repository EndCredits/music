## 关于 Alpha 版本

目前 R3PLAY 处于 Alpha 阶段，仍在开发中，功能尚未完善。建议每次更新时，先卸载旧版本的 R3PLAY，再重新安装。如遇到问题，欢迎提issue。

## ✨ 特性

- ✅ 使用 React + Electron 开发
- 🔴 网易云账号登录（扫码/手机/邮箱登录）
- 📺 支持 MV 播放
- 🚫🤝 无任何社交功能
- 🛠 更多特性开发中
